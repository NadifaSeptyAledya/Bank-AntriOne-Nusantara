# Bank AntriOne Nusantara

## Deskripsi
Aplikasi sistem antrean bank berbasis web yang digunakan untuk mengelola antrean nasabah secara online.

## Fitur
- Login Admin
- Login Petugas
- Login Nasabah
- Pengambilan nomor antrean
- Pemanggilan antrean
- Riwayat antrean

## Teknologi
- PHP
- MySQL
- HTML
- CSS

## Database
Import file `antrianbank.sql` ke phpMyAdmin sebelum menjalankan aplikasi.

## Cara Menjalankan
1. Simpan folder proyek di htdocs.
2. Jalankan Apache dan MySQL pada XAMPP.
3. Import database `antrianbank.sql`.
4. Buka localhost pada browser.
